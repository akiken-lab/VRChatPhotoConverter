# 実装タスク分解 v1.0

1. ソリューション構成を作成:
- `src/Core`
- `src/App.Wpf`
- `src/Infrastructure`
- `tests/Core.Tests`
2. 必要パッケージを追加:

- ロギング（Serilog相当）

## B. コア処理パイプライン
1. ファイルスキャナ実装:

2. コンバータ実装:
- PNG -> JPEG
- 品質設定
- 同名競合ポリシー
3. メタデータ引き継ぎ実装:

4. PNG保存処理実装:
- 移動/コピー
5. 実行サマリーモデルと構造化結果を実装。

## C. 監視機能（ゲーム終了トリガー）
1. 監視対象モデルを実装（`mode`, `exe_name`, `appid?`）。
2. プロセス監視を実装:
- `exe_name` による開始/終了検知
- 終了イベント発行
3. ジョブコーディネータを実装:
- 単一ワーカーロック
- 実行中の重複終了イベント無視
4. 同じコーディネータを使う手動実行エンドポイントを追加。
