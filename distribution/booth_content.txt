【VRC JPEG Auto Generator】VRChat 写真 JPEG 自動変換ツール

概要
- VRChat などで保存された PNG スクリーンショットを、JPEG にまとめて変換する Windows 用ツールです。
- ゲーム終了時の自動実行を主用途としており、タスクトレイ常駐にも対応しています。
- 初回ウィザード付きなので、入力フォルダと出力先を決めればすぐ使い始められます。

こんな人向け
- VRChat の写真を PNG のまま溜めがちで、JPEG 版を自動で作りたい
- ゲーム終了後にまとめて変換したい
- インストーラー不要のシンプルなツールを使いたい

主な機能
- PNG -> JPEG 自動変換
- ゲーム終了検知で自動実行
- トレイ常駐監視
- 手動実行（今すぐ変換）
- 起動時監視開始
- Windows 起動時の自動起動
- 重複変換防止の履歴 DB
- 元 PNG を残す / 削除する の切り替え
- 最小化時 / 閉じる時のトレイ格納通知

動作環境
- Windows 11 64bit で動作確認

動作確認済みゲーム
- VRChat

補足
- 監視対象 EXE を変更することで他ゲームへの流用も可能ですが、VRChat 以外は動作保証対象外です。

配布形式
- Portable 版（zip 配布）
- インストーラーはありません
- 解凍後に `VRCJpegAutoGenerator.exe` を実行してください

導入手順
1. zip を任意フォルダへ解凍
2. `VRCJpegAutoGenerator.exe` を起動
3. 初回ウィザードで入力フォルダと JPEG 出力先を設定
4. 必要に応じて「監視開始」「Windows 起動時に起動」を有効化

使い方
1. 監視を ON にして常駐します
2. VRChat 終了時に自動で変換されます
3. 必要なときは「今すぐ変換」で手動実行できます
4. 設定画面から JPEG 品質、除外秒数、ログレベルなどを調整できます

保存先
- 設定 / 履歴 / ログ: `%LOCALAPPDATA%\VRCJpegAutoGenerator`

注意事項
- `Delete` モードでは、JPEG 保存成功後に元 PNG を削除します
- 利用前に写真ファイルのバックアップを強くおすすめします
- 利用により発生した損害・データ損失等について作者は責任を負いません

ネットワーク・プライバシー
- 本ソフトは外部へのネットワーク通信を行いません
- テレメトリ、自動更新、広告表示はありません

監視負荷について
- 待機中の負荷は非常に低く、現在の一般的な PC 環境であれば通常利用ではほぼ無視できる水準です
- 通常は Windows のプロセス終了イベントを待ち受け、使えない環境のみ 2 秒ごとの軽い監視へ切り替えます
- 待機中にフォルダ全体の画像走査や変換処理を回し続ける設計ではありません
- そのため「常時重い監視」ではありませんが、厳密には負荷ゼロではありません

更新方法
- 新しい zip を展開し、既存の配布フォルダを置き換えてください
- 設定や履歴は `%LOCALAPPDATA%` 側に保存されるため、通常は引き継がれます
