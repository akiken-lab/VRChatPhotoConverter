VRC JPEG Auto Generator
=========================

概要
----
VRC JPEG Auto Generator は、ゲームで撮影された PNG を JPEG に自動変換する Windows 用ツールです。
監視常駐に対応しており、対象ゲーム終了時に自動実行できます。

主な機能
--------
- PNG -> JPEG 変換
- 監視常駐（トレイ常駐）
- 手動実行（今すぐ変換）
- Windows 起動時の自動起動（最小化起動）
- 実行履歴の保存（SQLite）

配布形式
--------
- Portable 配布（フォルダ配布）です。
- インストーラーは不要です。

インストール / 起動
------------------
1. zip 配下のフォルダ を任意の場所に解凍
2. VRCJpegAutoGenerator.exe を起動

データ保存先
------------
設定・履歴・ログは以下に保存されます。
%LOCALAPPDATA%\VRCJpegAutoGenerator

ネットワーク通信
----------------
- 本アプリは外部へのネットワーク通信を行いません。
- 利用状況テレメトリや自動更新機能はありません。
