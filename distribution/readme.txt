VRC JPEG Auto Generator
=======================

概要
----
VRC JPEG Auto Generator は、VRChat などで保存された PNG スクリーンショットを
JPEG に変換する Windows 用デスクトップアプリです。
ゲーム終了時の自動実行、トレイ常駐、手動実行に対応しています。

主な機能
--------
- PNG -> JPEG 変換
- ゲーム終了監視による自動実行
- タスクトレイ常駐
- 手動実行（今すぐ変換）
- 起動時に監視開始
- Windows 起動時の最小化起動
- 履歴 DB による再処理防止
- 最小化時 / 閉じる時のトレイ格納通知

使い方
------
1. 本フォルダを任意の場所へ展開します
2. VRCJpegAutoGenerator.exe を起動します
3. 初回ウィザードで入力フォルダと JPEG 出力先を設定します
4. 必要に応じて監視開始や Windows 起動時の起動を有効化します
5. メイン画面またはトレイメニューから「今すぐ変換」を実行できます

PNG の扱い
----------
- Keep: 元の PNG を残します
- Delete: JPEG 保存後に元の PNG を削除します

データ保存先
------------
設定・履歴・ログは配布フォルダではなく、以下に保存されます。
%LOCALAPPDATA%\VRCJpegAutoGenerator

主な保存内容:
- settings.json
- processing-history.db
- logs\app-YYYYMMDD.log

更新方法
--------
- 新しい zip を展開して配布フォルダを置き換えてください
- %LOCALAPPDATA% 配下の設定・履歴・ログはそのまま引き継がれます

アンインストール
----------------
1. 配布フォルダを削除
2. 完全削除したい場合のみ %LOCALAPPDATA%\VRCJpegAutoGenerator も削除

ネットワーク通信
----------------
- 本アプリは外部へのネットワーク通信を行いません
- テレメトリ、自動更新、広告表示はありません

監視負荷について
----------------
- 待機中の監視負荷は非常に低く、現在の一般的な CPU 環境であれば通常利用ではほぼ無視できる水準です
- 通常は WMI のプロセス終了イベントを購読し、使えない環境のみ 2 秒ごとの軽い監視へ切り替えます
- 待機中に画像変換やフォルダ全走査は行わず、変換処理は手動実行またはゲーム終了検知後にのみ走ります

注意事項
--------
- JPEG 変換および PNG 削除設定を含むため、定期的なバックアップを推奨します
- 利用により発生した損害・データ損失等について作者は責任を負いません
